# 📞 Sistema de Consulta de DDD do Brasil

Este projeto em C permite consultar informações de regiões do Brasil a partir do DDD (código de discagem direta a distância), exibindo cidade, estado e coordenadas geográficas.

---

## 🚀 Funcionalidades

- Busca de informações por DDD
- Exibição de cidade/região
- Exibição de estado (UF)
- Exibição de latitude e longitude
- Estrutura organizada com `struct`
- Código simples e fácil de expandir

---

## 💻 Tecnologias usadas

- Linguagem C
- Biblioteca padrão `stdio.h`

---

## 📌 Como funciona

O usuário digita um DDD e o sistema retorna:

- Região correspondente
- Estado (UF)
- Latitude
- Longitude
